How to Find a Steam AppID

This short guide explains how to find the Steam AppID (numeric ID) for any game or application using SteamDB.

Using SteamDB (recommended)

  1. Open https://steamdb.info.
  2. Use the search box at the top to type the game's name (for example: Portal 2).
  3. Click the game result. The URL will contain the AppID, for example: https://steamdb.info/app/620/ — here 620 is the AppID.
  4. Alternatively, on the game's Steam store page the AppID appears in the URL as well (e.g. https://store.steampowered.com/app/620/).

Notes and tips

  • Some apps (tools, middleware) may not appear in public lists — SteamDB shows public metadata and community data.
  • Use the numeric AppID when entering values on this site.
  • If you are using third-party tools, double-check the AppID before downloading manifests.

If you want, copy an example AppID and return to the Home page to check available manifests.
